## 1. 为什么不属于当前循环老化主线

该数据集为热失控传播实验——对电池单体/模组进行故意过热测试以研究故障级联行为。数据由滥用事件中多个热电偶的温度–时间曲线组成，不包含充放电循环数据，无容量衰减轨迹或周期级性能数据。

## 2. 包含的物理信息

- **热失控起始温度**：NMC-811 和 LFP 电芯自加热失控的临界温度。
- **传播动力学**：级联失效过程中相邻电芯之间的时间延迟和热传播路径。
- **峰值温度曲线**：热失控期间达到的最高温度（NMC 通常 700–1000°C）。
- **化学体系安全裕度对比**：相同滥用条件下 NMC-811（较低起始温度）与 LFP（较高起始温度）的热稳定性比较数据。
- **热释放速率**：失控事件中的热能输出，对热管理设计至关重要。

## 3. 未来可能支持的任务

- **安全感知数字孪生**：将热滥用极限纳入电池模型安全边界。
- **热管理系统设计**：用传播数据验证冷却策略和电芯间距设计。
- **故障预测**：识别失控前的早期热预警特征。
- **风险评分**：为跨化学体系的电池系统安全评级提供参考数据。
- **多物理场仿真验证**：在极端条件下校准电化学–热耦合模型。

## 4. 当前缺少的统一字段

| `cycle_index` | 不适用 | 无循环结构 |

| `voltage_v` | 部分 | 可能记录到电芯排气/短路前 |
| `current_a` | 不适用 | 无受控充放电 |
| `soh` | 不适用 | 不适用于滥用测试 |

| `propagation_delay_s` | 存在（非标准） | 电芯间失效时间差，需 schema 定义 |
