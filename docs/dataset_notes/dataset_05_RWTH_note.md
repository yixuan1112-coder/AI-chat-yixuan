## 数据格式

- **原始格式**: CSV (每个电池一个文件)
- **列**: time (s), current (A), voltage (V), temperature (°C)
- **大小**: 48 个文件, 总计数百 MB

## 处理注意事项

1. **电流符号约定**: 需逐文件验证。部分电池可能正值=放电。ETL 脚本通过电压-电流相关性自动检测。
2. **RPT过滤**: RPT 循环通过电压窗口超出正常范围 (V>4.0V 或 V<3.2V) 进行识别, 在 cycle_summary 中标记。
3. **SOH计算**: SOH = Q_discharge(第n圈) / Q_discharge(第1圈), 以第一个非 RPT 循环为参考。
4. **循环检测**: 每次放电开始 (电流由正/静置转为负值) 标记为新循环起点。
5. **C级电池**: 实际容量 (~1.85 Ah) 低于 A 级标称 (2.05 Ah), 因制造商分级所致; 所有电池来自同一生产批次, 保证一致性。

## 关键特点

- 48 颗电池使用**完全相同**的循环协议和温度 — 本数据集专为研究**电池间一致性/变异性**而设计。
- 部分循环协议 (3.5V–3.9V, ~20%–80% SOC) 模拟实际应用中电压限制的使用场景。
- 充放电量随老化略有变化 (内阻增大), 但相对老化容量的 DOD 基本恒定。

## 已知问题

- 测试批次启动时间略有错开 (相隔2–3天), 受设备限制。
- RPT 循环与常规循环交错排列, 分析时需分离。
- 周期性 RPT 不含阻抗数据 (仅 BOL 测试有)。
