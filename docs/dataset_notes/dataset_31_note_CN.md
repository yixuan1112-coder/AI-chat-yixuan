## 1. 为什么不属于当前循环老化主线

该数据集记录的是电池静置期间的电压弛豫曲线，而非连续充放电循环数据。不包含逐周期容量衰减轨迹或库仑效率趋势，无法映射到 BatteryTwin 循环老化管线的标准 `timeseries` 和 `cycle_summary` 表结构。

## 2. 包含的物理信息

- **开路电压（OCV）弛豫曲线**：电流中断后电压随时间的演化过程，反映电池内部电化学平衡过程。
- **SOC–OCV 对应关系**：弛豫终点电压近似各荷电状态下的平衡 OCV。
- **扩散动力学特征**：弛豫时间常数编码了固态锂扩散系数和电荷转移动力学信息。
- **多化学体系对比**：NMC811 和 LFP 的平行数据支持跨化学体系弛豫行为分析。

## 3. 未来可能支持的任务

- **基于静置电压的 SOH 估计**：利用弛豫曲线特征（斜率、稳定时间、终点电压）进行无损健康估计。
- **等效电路模型（ECM）参数辨识**：从弛豫瞬态响应中提取 R₀、R₁、C₁ 参数。
- **数字孪生状态初始化**：用弛豫数据校准物理模型的初始状态。
- **多模态融合**：将弛豫特征与循环数据结合，提升预后预测精度。

## 4. 当前缺少的统一字段

| 所需字段 | 状态 | 说明 |
|---------|------|------|
| `cycle_index` | 不适用 | 无循环结构，仅有静置段数据 |
| `capacity_ah` | 缺失 | 无放电容量测量 |
| `coulombic_efficiency` | 不适用 | 弛豫数据无此概念 |
| `soh` | 缺失 | 无参考容量用于 SOH 计算 |
| `temperature` | 待确认 | 是否记录了温度需下载后核实 |
| `current_a` | 部分 | 弛豫期间电流名义为零；静置前的电流可能未记录 |
