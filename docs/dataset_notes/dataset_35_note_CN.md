## 1. 为什么不属于当前循环老化主线

该数据集为加速量热仪（ARC）测量数据——通过控制慢速升温表征电芯自加热起始和热失控行为。数据为绝热条件下的温度–时间曲线，非充放电循环。每颗电芯经历单次破坏性热测试，无容量或循环指标。

## 2. 包含的物理信息

- **自加热起始温度**：放热反应开始、电芯不再被认为热稳定的临界温度。
- **放热速率曲线（dT/dt）**：温升速率随温度的变化，揭示 SEI 分解、负极–电解液反应和正极分解的反应动力学。
- **总热释放量**：从热失控起始到完成的累积热能输出。
- **多化学体系对比**：相同 21700 规格下 NMC、NCA 和 LFP 的 ARC 平行数据，可直接进行热安全排序。
- **绝热热失控轨迹**：最坏工况（无冷却）下的完整温度演化过程。

## 3. 未来可能支持的任务

- **化学体系安全排序**：定量比较不同正极化学体系的热稳定性，辅助电芯选型。
- **热模型参数化**：ARC 导出的反应动力学参数（Arrhenius 参数）用于热–电化学耦合模型。
- **电池安全认证支持**：ARC 数据是 UN 38.3 和 IEC 62660-3 安全评估的标准输入。
- **数字孪生安全包络**：在电池管理系统中定义基于温度的安全边界和触发条件。
- **梯次利用筛选**：利用热稳定性指标评估老化电芯的再利用适宜性。
