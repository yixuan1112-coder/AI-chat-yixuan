## 1. 为什么不属于当前循环老化主线

该数据集为电化学阻抗谱（EIS）频率扫描数据，而非充放电循环记录。数据结构为频率–阻抗对（Nyquist/Bode 格式），与循环时序数据的电压–电流–时间结构本质不同。不提供逐周期容量或效率指标。

## 2. 包含的物理信息

- **阻抗谱（Nyquist 图）**：在激励频率范围内的实部和虚部阻抗分量，编码了内阻和电荷转移行为。
- **欧姆电阻（R₀）**：高频截距，反映电解液和接触电阻。
- **电荷转移电阻（Rct）**：中频半圆直径，反映电极反应动力学。
- **Warburg 扩散**：低频尾部特征，捕捉固态离子扩散限制。
- **SEI 层表征**：阻抗特征与固体电解质界面膜生长相关。

## 3. 未来可能支持的任务

- **基于阻抗的 SOH 诊断**：跟踪电阻增长作为无损健康指标。
- **老化机制辨识**：分离欧姆、动力学和扩散对老化的贡献。
- **快速筛选/质量控制**：EIS 作为电芯分级和梯次利用评估的快速诊断手段。
- **物理信息模型验证**：用实测阻抗谱校准电化学模型（P2D、SPM）。
- **多模态数字孪生**：将 EIS 快照与循环历史融合，实现综合状态估计。

## 4. 当前缺少的统一字段

| 所需字段 | 状态 | 说明 |
|---------|------|------|
| `cycle_index` | 不适用 | 无循环数据 |
| `capacity_ah` | 缺失 | 无容量测量 |
| `voltage_v` / `current_a` | 不适用 | EIS 使用交流微扰，非直流充放电 |
| `soh` | 缺失 | 无参考容量 |
| `temperature` | 待确认 | 可能作为测试条件记录 |
| `frequency_hz` | 存在（非标准） | EIS 核心字段，当前 BatteryTwin schema 未定义 |
| `z_real` / `z_imag` | 存在（非标准） | EIS 核心字段，需新增 schema 定义 |
