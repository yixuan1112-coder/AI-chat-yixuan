## 1. 为什么不属于当前循环老化主线

该数据集记录机械滥用测试（针刺实验），故意触发内短路和热失控。数据为每颗电芯的单次破坏性事件，非重复充放电循环。不包含容量衰减或老化轨迹。

## 2. 包含的物理信息

- **机械失效阈值**：内短路触发时的力、位移和变形数据。
- **内短路（ISC）特性**：针刺引发内短路时的电压降速率和电流尖峰特征。
- **机械滥用的热响应**：从 ISC 触发到热失控全过程的温度演化。
- **失效模式分类**：根据穿刺深度和速度区分温和 ISC（可恢复电压跌落）和灾难性失控。

## 3. 未来可能支持的任务

- **碰撞安全建模**：为电动车碰撞仿真和电池包设计提供机械–热失效数据。
- **滥用耐受基准测试**：量化不同电芯规格和化学体系的机械鲁棒性。
- **ISC 故障注入模型**：用针刺数据校准数字孪生安全模块的内短路模型。
- **早期故障检测**：将机械变形特征与电气异常关联，用于实时 BMS 故障检测。

## 4. 当前缺少的统一字段

| 所需字段 | 状态 | 说明 |
|---------|------|------|
| `cycle_index` | 不适用 | 单次破坏性事件，无循环 |
| `capacity_ah` | 不适用 | 测试中电芯被破坏 |
| `soh` | 不适用 | 不适用 |
| `temperature` | 存在 | 滥用事件期间的热电偶数据 |
| `voltage_v` | 存在 | 记录到电芯失效为止 |
| `force_n` / `displacement_mm` | 存在（非标准） | 机械加载数据，需新增 schema 字段 |
| `penetration_depth_mm` | 存在（非标准） | 针刺参数，需 schema 定义 |
| `isc_trigger_time` | 存在（非标准） | 内短路触发时间戳 |
